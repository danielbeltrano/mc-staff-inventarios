# Campus virtual para el personal de la institución
Este es un sistema de gestión integral para instituciones educativas, que permite administrar todas las funciones académicas y administrativas, así como facilitar la interacción entre padres, estudiantes y la institución.

## Instalación

Para instalar y ejecutar este proyecto localmente, sigue estos pasos:

Instala las dependencias:
npm install

Ejecuta el proyecto:
npm start

## Características principales

- Autenticación de usuarios con **Supabase**.
- Gestión revisión y aprobación de las matrículas de los estudiantes y gestión y creación de usuarios con rol.
- Carga y gestión de información documentos a través de **Supabase Storage**.
- Check de aprovación de la información y documentos de los estudiantes.
- Rutas protegidas para garantizar que solo los usuarios autenticados puedan acceder a ciertas páginas.
- Diseño responsive con **Tailwind CSS** para una experiencia de usuario óptima en diferentes dispositivos.

## Tecnologías Utilizadas

- **Frontend**:
  - React.js
  - Redux Toolkit
  - React Router DOM
  - Tailwind CSS

Para trabajar sobre el proyecto sigue estos pasos:

Haz un fork del repositorio.
1. Crea una rama con una nueva funcionalidad (git checkout -b feature/nueva-funcionalidad).
2. Haz commit de tus cambios (git commit -m 'Añadir nueva funcionalidad').
3. Haz push a la rama (git push origin feature/nueva-funcionalidad).
